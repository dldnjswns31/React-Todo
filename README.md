# Todo List with React

## 📎 배포 링크

[바로 가기](https://react-todo-eight-taupe.vercel.app/)

## ⚙️ 실행 방법

```
# 실행 전 패키지를 설치합니다.
npm install

# 서버를 실행합니다.
npm start
```
